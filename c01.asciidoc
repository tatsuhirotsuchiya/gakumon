==== Hello World!

最初のプログラムでは，Hello World!を表示させるのが定番．まずは，次のプログラムを実行しよう．

////

最初のプログラムは，Hello World!を表示させるものと決まっています．まずは，次のプログラムを実行してみましょう．

まず，エディタを起動して，プログラムを書きます．エディタは文字だけからなるデータを編集するためのツールで，Windowsならメモ帳，Macなら．．といったソフトウェアが相当します．打ち終えたらファイルとして保存しましょう．ここでファイル名は`hoge.py`としておきます．


次に，ターミナルという，直接文字を入力してコンピュータに様々な処理をさせたり，処理された結果を文字として表示するためのツールを起動します．
////

* 名前を付けて保存

==== セルシウス度とファーレンハイト度

セルシウス度からファーレンハイト度への変化を考える．
-273.15度より低い温度は存在しないので，無視しよう．
これは，ifを使って実現できる．

.temparature.py
[source,python]
----
# セルシウス度->ファーレンハイト度
print("温度を入力してください")
x = input()
x = float(x)
print("セルシウス度:", x)
if x >= -273.15:
    y = 1.8*x + 32
    print("ファーレンハイト度:", y)
----

-273.15度より低い値が入力されたとき，何かメッセージを出すように，プログラムを拡張しよう．if-elseをつかえばよい．

.temparature.py
[source,python]
----
# セルシウス度->ファーレンハイト度
print("温度を入力してください")
x = input()
x = float(x)
print("セルシウス度:", x)
if x >= -273.15:
    y = 1.8*x + 32
    print("ファーレンハイト度:", y)
else:
    print("温度が低すぎます")
----

温度の対応表を出力してみる．
いろいろな対象に対して，処理を繰り替えすforがつかえる．

.temparature2.py
[source,python]
----
# セルシウス度->ファーレンハイト度
for i in range(-5, 40):
    print(i, "->", 1.8*i + 32)
----

==== 入れ子 (nesting)

ifやforなどの構造は，入れ子にできる．

.abc.py
[source,python]
----
# a, b, c どれが一番大きい？
print("a?")
a = input()
print("b?")
b = input()
print("c?")
c = input()

if a > b:
    if a > c:
        print(a)
    else:
        print(c)
else:
    if b > c:
        print(b)
    else:
        print(c)
----

=== 課題
FizzBuzzは，ひとりずつ１から順に数を数えていって，3で割り切れる場合は数ではなく「Fizz」，5で割り切れる場合は「Buzz」、両者で割り切れる場合は「Fizz-Buzz」と言うゲームである．これを，50までの数に対して行うプログラムをつくろう．

数のあまりは，%をつかって計算できる．たとえば，5 % 2 は 1．

二つの数が等しいかどうかは， == をつかって判定する．a == b は，aとbが同じ値なら True (真) となり，そうでなければ False (偽) となる．

# これは数あてゲームです
import random

number = random.randint(1, 20)

print('この数は，1から20までの整数です')

for i in range(5):
    print('数をあててみて？')
    guess = input()
    guess = int(guess)

    if guess < number:
        print('小さすぎ')
    elif guess > number:
        print('大きすぎ')
    else:
        break

if guess == number:
    print('あたり！ トライした回数は' + str(i+1) + '回でした')
else:
    print('残念: 答えは' + str(number) + 'でした')
