==== セルシウス度とファーレンハイト度

セルシウス度からファーレンハイト度への変化を考える．
-273.15度より低い温度は存在しないので，無視しよう．
条件が成り立つときだけ実行したい処理は，ifを使って書く．
この処理の部分は，半角スペースで4文字下げること．


.temparature.py
[source,python]
----
# セルシウス度->ファーレンハイト度
print("温度を入力してください")
x = input()
x = float(x)
print("セルシウス度:", x)
if x >= -273.15:
    y = 1.8*x + 32
    print("ファーレンハイト度:", y)
----

-273.15度より低い値が入力されたとき，何かメッセージを出すように，プログラムを拡張しよう．
条件が成り立たない処理も書きたいときは，if-elseをつかえばよい．

.temparature.py
[source,python]
----
# セルシウス度->ファーレンハイト度
print("温度を入力してください")
x = input()
x = float(x)
print("セルシウス度:", x)
if x >= -273.15:
    y = 1.8*x + 32
    print("ファーレンハイト度:", y)
else:
    print("温度が低すぎます")
----

温度の対応表を出力してみる．
いろいろな対象に対して，処理を繰り替えす `for` がつかえる．

.temparature2.py
[source,python]
----
# セルシウス度->ファーレンハイト度
for i in range(-5, 40):
    print(i, "->", 1.8*i + 32)
----

`range(-5,40)` は-5, -4, ..., 39をあらわす．
 `i` にそれぞれの値を入れて，ループを繰り返す．
ループの部分は，4文字下げる．

==== 入れ子 (nesting)

ifやforなどの構造は，入れ子にできる．

.abc.py
[source,python]
----
# a, b, c どれが一番大きい？
print("a?")
a = int(input())
print("b?")
b = int(input())
print("c?")
c = int(input())

if a > b:
    if a > c:
        print(a)
    else:
        print(c)
else:
    if b > c:
        print(b)
    else:
        print(c)
----

=== 課題
FizzBuzzは，ひとりずつ１から順に数を数えていって，3で割り切れる場合は数ではなく「Fizz」，5で割り切れる場合は「Buzz」、両者で割り切れる場合は「Fizz-Buzz」と言うゲームである．これを，50までの数に対して行うプログラムをつくろう．

数のあまりは，%をつかって計算できる．たとえば，5 % 2 は 1．

2つの数が等しいかどうかは， == をつかって判定する．a == b は，aとbが同じ値なら True (真) となり，そうでなければ False (偽) となる．

.注意
* 行頭の空白には意味がある．その行がどこの階層にあるかを示している．
1つの階層を4つの半角の空白文字(スペース)であらわすこと．
* = は，左に右を代入するという意味．等号ではない！
等号は ==．不等号は !=．

==== ヒント
まずは，1から50までを表示．次に，3の倍数の時に「Fizz」を表示，というように，少しずつ改良していこう．

.fizz.py
[source,python]
----
for i in range(1,51):
    if i % 3 == 0:
        print("Fizz")
    else:
        print(i)
----
